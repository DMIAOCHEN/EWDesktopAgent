# Quickstart: 智能桌面客户端

## 安装与启动

### 环境要求

- 操作系统: Windows 10/11 (64位)
- 内存: 至少 2GB 可用
- 磁盘: 至少 500MB 可用空间
- 网络: 需要连接内网业务系统和FastGPT服务

### 安装步骤

1. 下载安装包 `EWDesktopAgent-Setup-x.x.x.exe`
2. 运行安装程序
3. 跟随安装向导完成安装
4. 启动客户端

### 首次登录

1. 启动客户端后，输入用户名和密码
2. 首次登录需要绑定设备
3. 登录成功后，加载业务系统列表

---

## 核心功能使用

### 1. 访问业务系统

1. 在左侧系统列表中点击要打开的业务系统(如RIS)
2. 系统自动在新标签页中打开
3. 如需打开多个系统，点击"+"号创建新标签页

**快捷键**:
- `Ctrl+T`: 新建标签页
- `Ctrl+W`: 关闭当前标签页
- `Ctrl+Tab`: 切换到下一个标签页

### 2. 使用智能助手

**唤起方式**:
- 点击右下角助手图标
- 使用快捷键 `Ctrl+Space`
- 语音唤起 (如果已启用)

**使用示例**:
```
用户: 帮我查一下患者王建的CT检查
助手: 正在为您搜索...
(助手自动在当前页面输入"王建 CT"并搜索)
```

### 3. 配置智能提醒

1. 点击右上角设置图标
2. 进入"提醒"页面
3. 点击"新建提醒"
4. 配置提醒条件:
   - 触发类型: 时间/事件/API轮询
   - 提醒内容: 可使用变量如 `{count}`, `{patient_name}`
   - 目标页面: 选择要跳转的业务系统页面

### 4. 语音交互

1. 首次使用需要授权麦克风
2. 按住麦克风按钮说话
3. 松开按钮后自动识别
4. 识别结果自动执行

**示例**:
```
用户: (按住说话) 帮我查看今天的手术安排
系统: 正在为您打开手术室系统...
(自动导航到手术排班页面)
```

### 5. 个性化学习

客户端会自动学习您的使用习惯:
- 常用系统
- 工作时间规律
- 查询偏好

**管理个性化设置**:
1. 进入"设置" → "隐私"
2. 可以查看学习的数据
3. 可以关闭个性化学习
4. 可以清除学习数据

## 常见问题

### Q: 打开业务系统很慢怎么办?
A: 检查网络连接。可以尝试刷新页面或关闭不用的标签页释放内存。

### Q: 智能助手理解错了我的指令怎么办?
A: 可以点击"重新输入"或切换到文字输入。助手会记住上下文并改进。

### Q: 语音识别不准怎么办?
A: 确保麦克风正常，选择安静环境。可以在设置中调整语音识别灵敏度。

### Q: 内存占用过高怎么办?
A: 关闭不用的标签页，减少同时打开的业务系统数量。

### Q: 需要离线使用可以吗?
A: 部分功能支持离线: 浏览已缓存页面、查看本地提醒记录、离线设置的提醒规则。
